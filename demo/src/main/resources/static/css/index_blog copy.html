Bem-vindo ao Salve!

By Ian Almeida - VP Design under CSS Pure

A plataforma Salve! busca trazer multiplos ambientes de produção ou implementações, a depender das nescessidades dos clientes, onde cada um possa ter o seu produto de forma modular e prática para cada empreendimento. Nossa plataforma de uso educacional é a plataforma de integração modular carro chefe dessa companhia. As nosso mais novo produto está preste a ser lançado nossa plataforma voltada para as nescessidades hospitalares, com toda uma rede de gerenciamento controle e fluxo do estoque.
